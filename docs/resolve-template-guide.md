# DaVinci Resolve Template Guide

This document explains how to create a reusable template project for your game review videos. Once built, you can duplicate the project and replace the placeholder clips with your own footage.

## 1. Create the Horizontal (Long‑Form) Template

1. Open DaVinci Resolve → **New Project** → name it `Review_Longform_Template`.
2. Go to **File → Project Settings → Master Settings**:
   - Timeline resolution: **2560 x 1440** (or 1920x1080 if you prefer)
   - Frame rate: **match your typical recording** (24, 30, or 60 fps)
3. In the **Edit** page, build the following track layout:

- Caption style: larger font (~80), positioned near the bottom.
- Export as `Review_Shorts.drp`.

## 3. Using the Templates

- Duplicate the `.drp` file for each new review.
- Replace the placeholder clips with your cleaned gameplay footage and voiceover.
- Import the `captions.srt` generated by the pipeline (Timeline → Import → Subtitle).
- Adjust the music/SFX to taste.
- Render.

Once you’ve made the templates, copy the `.drp` files into `templates/davinci/` in the repo.
